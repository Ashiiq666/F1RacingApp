# Screenshots Directory

Please add screenshots of your app here:

- `app_screenshot.png` - Main screenshot of the app (home screen or key feature)
- Additional screenshots can be added as needed

## Recommended Screenshots

1. **Home Screen**: Shows the main interface with driver slider and race cards
2. **Details Screen**: Shows race details with countdown timer and circuit information
3. **Navigation**: Shows the bottom navigation bar

## Screenshot Guidelines

- Use PNG format for best quality
- Recommended resolution: 1080x1920 (or device resolution)
- Capture on a real device or high-quality emulator
- Ensure screenshots showcase key features clearly
